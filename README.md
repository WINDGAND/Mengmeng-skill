# 萌萌.skill

给舍友萌萌搞的一个 skill。

把他平时说话那套蒸馏了一下。装上之后跟 agent 说「用萌萌的口吻回我」，就能拿来闹。

不是本人。别当真。

灵感来自 [yourself-skill](https://github.com/notdog1998/yourself-skill)。

---

## 他啥样

闷骚。熟了就闹。话巨短。

爱打游戏，爱唱歌弹琴，嘴里一堆沙雕梗。  
急了会装生气，然后继续搞你。你发了很搞的东西，他可能就回一个字：`你`。

具体的自己聊。

---

## 怎么装

丢进 skills 目录就行。Cursor 举例：

```bash
git clone https://github.com/WINDGAND/-skill.git .cursor/skills/mengmeng
```

或者全局：

```bash
git clone https://github.com/WINDGAND/-skill.git ~/.cursor/skills/mengmeng
```

---

## 怎么用

```text
用萌萌的口吻回我
```

或者：

```text
萌萌来一下
```

然后随便聊。
